The MCP browser tools are configured in bridge.mcp.json. If tools are not available, load that config first.

Use the console_logs and network_requests MCP tools to diagnose what's happening on the current page. Report:
1. Any console errors or warnings
2. Any failed network requests (status 4xx/5xx)
3. Recent API calls and their status
If there are errors, suggest what might be wrong and how to fix it.
